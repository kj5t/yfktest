<h2 class="western"><a href="http://www.drdobbs.com/open-source/why-i-use-perland-will-continue-to-do-so/240148364">yfktest
- Contest Logging
Software PROUDLY written in PERL</a></h2>
<p>YFKtest is a logbook program for&nbsp;<a href="http://en.wikipedia.org/wiki/Contesting">ham
radio contests</a>. It is easily extendable to log and score
arbitrary contests. It doesn't require any installation or setup,
just download and start!</p>
<p>First written in 2007 by Fabian, DJ1YFK. Since 2011 the coding is
being done by Bob, W9YA. It's a FB logger and a great way to study <a href="http://en.wikipedia.org/wiki/Perl">perl</a> coding. Many users prefer it's <a href="http://en.wikipedia.org/wiki/KISS_principle">K.I.S.S.</a> logging style. 
</p>

<h2 class="western"><a href="https://git.fkurz.net/yfktest/yfktest/src/branch/master/docs/MANUAL">Key Features</a></h2>
<ul>
	<li><p style="margin-bottom: 0in">Simple console based user
	interface, similar to <a href="http://www.k1ea.com/">CT</a> or <a href="http://www.ei5di.com/">SD</a>
	.</p>
	</li><li><p style="margin-bottom: 0in">Uses K5ZD's <a href="http://www.supercheckpartial.com/">super
	partial check</a> files (included). These can be extended with your own file of "friends" .</p>
	</li><li><p style="margin-bottom: 0in">CW messages via <a href="http://cwdaemon.sourceforge.net/">Cwdaemon</a>
	by PG4I (for parallel and pin-controllable rs-232 ports) OR
	<a href="http://home.kpn.nl/w.knol18/qle/winkeyer/index.html">winkeydaemon.pl</a>
	(version 1.0.PE7T-2) by PE7T (for usb and other software-controlled
	serial ports). BOTH files are now included ! 
	</p>
	</li><li><p style="margin-bottom: 0in"><a href="http://www.adif.org/">ADIF</a>
	and <a href="http://www.kkn.net/~trey/cabrillo/">Cabrillo</a>
	export, PLUS generation of summary files . 
	</p>
	</li><li><p style="margin-bottom: 0in">Rig control via <a href="http://sourceforge.net/projects/hamlib/">hamlib</a>
	.</p>
	</li><li><p style="margin-bottom: 0in">Very easy and completely menu
	driven setup . 
	</p>
